# MERN Game Hub

A simple MERN stack application that displays games as cards and includes a founding team section.

## Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- CSS (Flex layout)

## Features
- Game cards (4x2 layout)
- Play button
- Founders section with photos
- REST API for games

## Run Locally

### Server
cd server  
npm install  
npm run dev  

### Client
cd client  
npm install  
npm run dev
